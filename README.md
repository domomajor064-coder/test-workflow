# Test Workflow Project

A simple Python project demonstrating the dual worker workflow (Minimax Developer + Kimi Reviewer).

## Overview

This project is a minimal test to verify the dual worker workflow is functioning correctly. It includes:

- `hello.py` - A simple Python script that prints a greeting
- `README.md` - This file

## Usage

Run the greeting script:

```bash
python3 hello.py
```

## Testing

The script should output:

```
Hello from Minimax Worker!
```

## Development Workflow

1. **Minimax Developer** - Implements the solution
2. **Kimi Reviewer** - Reviews the code

## Requirements

- Python 3.6 or higher
