# Test Workflow Project

A simple Python project that demonstrates the dual worker workflow (Minimax Developer + Kimi Reviewer).

## Quick Start

```bash
python hello.py
```

## Project Structure

- `hello.py` - Main script that prints a greeting
- `DESIGN.md` - Project requirements and acceptance criteria

## What This Does

This project is a minimal test to verify the dual worker workflow is functioning correctly. The workflow consists of:

1. **Minimax Developer** - Implements features and creates Pull Requests
2. **Kimi Reviewer** - Reviews PRs and provides feedback

## Running the Script

Simply run:

```bash
python hello.py
```

Expected output:
```
Hello from Minimax Worker!
```
